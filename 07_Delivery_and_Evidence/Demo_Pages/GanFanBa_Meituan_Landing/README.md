# 干饭吧美团转化落地页

本目录保留一版公开可展示的活动转化落地页，用于说明“玩法决策 -> 结果展示 -> 外部优惠承接”的链路设计。

## 页面目标

- 降低用户“不知道吃什么”的决策门槛。
- 用抽签玩法承接广告/活动流量。
- 在结果页提供外部优惠入口，形成从内容互动到转化跳转的闭环。

## 公开版处理

- 真实美团渠道链接已替换为 `YOUR_MEITUAN_DEEPLINK` 和 `YOUR_MEITUAN_H5_FALLBACK_URL`。
- 不包含服务器路径、部署账号、统计平台密钥或真实投放参数。
- 页面仅作为作品集演示材料，不代表线上最终版本。

## 可展示能力

- 活动页信息架构
- 移动端互动页面实现
- 外跳转化路径设计
- 埋点事件预留
