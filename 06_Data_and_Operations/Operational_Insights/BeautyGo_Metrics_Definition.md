# 核心指标口径

## 1. 线索类指标

### 新增客户线索

- 定义：当周新提交且联系方式有效的客户预约数
- 不计入：重复提交、明显无效线索、非徐州线索

### 有效客户线索

- 定义：满足城市、场景、时间、联系方式四个条件的线索数
- 用途：判断获客质量，而不是单看流量

### 新增化妆师申请

- 定义：当周新提交招募表单的化妆师数

### 合格化妆师

- 定义：作品可用、价格合理、可接试运营单、愿意配合规则的化妆师数

## 2. 交易类指标

### 撮合单数

- 定义：客户需求与至少 1 位化妆师进入实质性报价和确认流程的订单数

### 已支付订单数

- 定义：已支付定金或全款，且有支付记录的订单数

### 已完成订单数

- 定义：服务已履约结束，且客户未提出严重争议的订单数

### 完单率

- 公式：`已完成订单数 / 已支付订单数`

### 客单价

- 公式：`支付总金额 / 已支付订单数`

## 3. 质量类指标

### 平均满意度

- 公式：所有已收回评价分数的平均值
- 建议采用 5 分制

### 复购意向数

- 定义：明确表达“愿意再次使用”或“愿意推荐给朋友”的用户数

### 取消订单数

- 定义：已进入匹配或支付流程，但最终未履约的订单数

### 取消原因

统一收敛为以下枚举：

- 时间不合适
- 价格不接受
- 距离不合适
- 化妆师拒单
- 客户改计划
- 沟通不顺畅
- 对效果不放心
- 其他

## 4. 周复盘必须回答的问题

1. 线索少，还是有效线索少。
2. 需求有，但供给接不住，还是供给有，但用户不付费。
3. 最大损耗发生在：
   留资、沟通、报价、支付、履约还是回访。
4. 用户和化妆师各自最大的异议是什么。
5. 下一周应该补渠道、补供给、调价格，还是调规则。
