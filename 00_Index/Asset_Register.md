# 资产登记表

本文件用于记录放入仓库的公开材料。上传前应确认材料已经脱敏，并删除密钥、账号、客户隐私、原始商业数据和内部专用信息。

| 资产类型 | 建议目录 | 收录标准 |
| --- | --- | --- |
| PRD / 产品方案 | `01_Product_Strategy/PRD/` | 能说明问题定义、用户场景、功能边界和决策依据 |
| 竞品 / 市场分析 | `02_Market_and_User_Insights/Competitive_Research/` | 能说明机会判断、市场比较和产品定位 |
| 用户流程 / 原型 | `03_User_Experience/` | 能说明体验路径、信息架构和关键交互 |
| 系统结构 / 后台逻辑 | `04_System_Architecture/` | 能说明前后台协作、数据流和模块关系 |
| AI 工作流 | `05_AI_Agent_Workflows/` | 能说明多智能体、提示词或自动化流程设计 |
| 数据与运营材料 | `06_Data_and_Operations/` | 能说明数据采集、清洗、匹配和运营支撑 |
| 截图 / 演示材料 | `07_Delivery_and_Evidence/` | 能证明阶段性交付、展示和归档能力 |
| 脚本 / 工具 | `08_Scripts_and_Tools/` | 能证明通过技术手段提升执行效率 |

## 不应提交的内容

- 完整原始压缩包
- 账号、密码、Token、Access Key
- 未脱敏的客户或商家明细
- 构建产物、依赖目录、缓存文件
- 仅供内部使用且不可公开的材料
