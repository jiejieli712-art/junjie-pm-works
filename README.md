# 产品作品集

这是一份经过整理的产品作品集，记录我在产品策略、系统思考、用户体验、AI 工作流探索和项目落地中的核心产出。

我希望这个仓库呈现的不是文件数量，而是思考方式。  
它用于展示我如何理解问题、组织复杂信息，并将模糊需求转化为可讨论、可设计、可落地的产品方案。



## 关于我

我是一名关注业务逻辑、用户体验与 AI 工具结合的产品创造者。

我的工作习惯介于产品规划与技术落地之间。我关注一个产品为什么值得做，系统应该如何运转，用户体验应该如何被组织，以及 AI 工具如何帮助团队更高效地从想法走向结构化产出。

## 核心能力

### 产品规划

我擅长将模糊机会拆解为结构化需求、用户场景、功能边界和可执行文档。

能力关键词：

- PRD 梳理
- 产品范围定义
- 用户场景拆解
- 业务逻辑建模
- 路线图与交付规划



### AI Agent 编排

我关注如何通过 AI Agent、提示词和自动化流程降低重复劳动，并提升产品执行效率。

能力关键词：

- 多智能体工作流设计
- Prompt 策略
- 人机协同流程设计
- 自动化机会识别
- AI 辅助研究与文档生成



### 用户体验

我偏好克制、清晰、可用的界面体验。相比装饰性表达，我更重视层级、路径、状态和交互逻辑。

能力关键词：

- 用户流程设计
- 信息架构
- 原型评审
- 界面行为拆解
- 克制且有秩序的视觉判断
