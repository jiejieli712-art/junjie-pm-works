# 推广采集助手

推广采集助手是一个 Tampermonkey 用户脚本，用于辅助采集饿了么联盟推广页面中的店铺列表和推广链接，并将结果导出为 CSV 或 JSON 文件。

当前版本：`1.3.3`

## 主要功能

- 采集当前列表页中的店铺信息。
- 逐个打开店铺推广详情并采集推广链接。
- 支持按市、区、县批量采集。
- 支持输入多个区域并依次处理。
- 当区域店铺数量较多时，按类目补充采集。
- 自动恢复城市筛选、类目筛选和分页状态。
- 将采集结果导出为 CSV 或 JSON 文件。
- 对 `localStorage` 中的数据进行分级压缩，降低长时间采集时的存储压力。

## 安装方法

1. 在浏览器中安装 [Tampermonkey](https://www.tampermonkey.net/) 扩展。
2. 打开仓库中的 [`promotion-capture-helper.user.js`](./promotion-capture-helper.user.js)。
3. 将脚本内容复制到 Tampermonkey 的新建脚本页面。
4. 保存脚本并确保脚本已启用。
5. 访问饿了么联盟推广页面，页面右侧会显示“推广采集助手”面板。

## 使用方法

面板支持以下操作：

- `批量区域转链`：按输入的多个区域依次采集推广链接，并为每个区域自动导出文件。
- `批量区域店铺`：按输入的多个区域依次采集店铺列表，并为每个区域自动导出文件。
- `采集首个区域转链`：采集输入框中第一个区域的推广链接。
- `采集首个区域店铺`：采集输入框中第一个区域的店铺列表。
- `采集下一家`：采集当前页面中的下一家待处理店铺。
- `停止采集`：请求停止正在执行的采集流程。
- `导出CSV`：导出当前暂存数据。
- `导出JSON`：导出当前暂存数据和日志。
- `清空数据`：清空脚本暂存的采集数据。

区域输入框支持以下格式：

```text
徐州市-泉山区
徐州市：云龙区、泉山区、铜山区
云龙区
泉山区
```

## 导出文件

批量模式会按区域自动生成文件，例如：

```text
徐州市-泉山区.csv
徐州市-泉山区.json
```

CSV 文件适合直接使用表格软件打开。JSON 文件会额外保留店铺数据、详情数据和采集日志。

## 注意事项

- 本脚本仅用于辅助操作，运行前请确认目标页面已经登录并可正常访问。
- 页面结构或接口发生变化后，脚本可能需要同步更新。
- 批量操作会自动翻页和打开详情页，请避免在采集过程中手动切换页面。
- 请遵守目标平台的使用规则，并仅在获得授权的场景中使用采集功能。
