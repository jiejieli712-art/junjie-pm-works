# 联盟推广采集助手脚本

本目录收录两个 Tampermonkey / 油猴自动化脚本，用于在已登录的联盟后台页面中辅助采集活动推广物料，并导出 CSV / JSON。

## 文件说明

| 文件 | 版本 | 适用页面 | 主要用途 |
| --- | --- | --- | --- |
| `Meituan_Activity_Promotion_Collector_v1.1.1.user.js` | 1.1.1 | 美团联盟「物料推广 > 活动推广」 | 批量采集活动基础信息、推广媒体/推广位下生成的文案素材和链接。 |
| `Taobao_Shangou_Venue_Promotion_Collector_v1.0.4.user.js` | 1.0.4 | 饿了么 / 淘宝闪购联盟会场推广页 | 批量采集会场活动信息，进入推广详情页后采集不同推广链路下的文案、链接和图片地址。 |

## 功能摘要

- 基于当前已登录页面执行 DOM 自动化，不调用第三方接口。
- 支持暂停、继续、停止、清空缓存。
- 支持 localStorage 缓存和成功记录去重。
- 支持导出 CSV / JSON。
- CSV 导出已处理中文表头和 BOM，便于表格软件打开。
- 脚本面板支持拖拽和最小化，便于长时间采集时观察页面状态。

## 安装方法

1. 在浏览器中安装 Tampermonkey / 油猴扩展。
2. 在扩展中新建脚本。
3. 复制对应 `.user.js` 文件内容并保存。
4. 打开对应联盟后台页面，确认已登录且页面正常加载。
5. 使用页面右下角悬浮面板开始采集。

## 使用注意

- 脚本只用于辅助前端页面操作，不绕过登录、权限、验证码或风控。
- 采集过程中不要手动切换页面或修改筛选条件。
- 如果平台页面结构调整，按钮、下拉框或详情页字段可能需要同步维护。
- 公开展示前应确认导出的业务数据、链接和账号信息已按需脱敏。
