# 数理宝典代码与架构概览

## 项目定位

数理宝典是一个面向中考数学复习的离线学习 App，当前工程基于 `uni-app + Vue 3 + TypeScript + Pinia`。项目重点是用低成本方式组织公式知识库、复习状态和闯关练习，适合 H5 预览、App 构建和本地 APK 演示。

## 技术栈

- 跨端框架：`uni-app`
- 前端框架：`Vue 3`
- 类型系统：`TypeScript`
- 状态管理：`Pinia`
- 检索辅助：`pinyin-pro`
- 构建工具：`Vite`

## 核心目录

| 目录 | 作用 |
| --- | --- |
| `data/` | 本地公式库、闯关题库等结构化数据 |
| `pages/` | 主要业务页面实现 |
| `src/pages/` | 页面包装入口，用于对齐 uni-app 构建入口 |
| `components/` | 公式卡、底部导航、闯关组件等通用组件 |
| `stores/` | 收藏、重点、复习进度和闯关状态管理 |
| `utils/` | 导航、搜索、数据处理等工具函数 |
| `docs/` | 页面入口约定、Android 构建说明和手工冒烟清单 |
| `scripts/` | HBuilderX Node 兼容、APK 构建等辅助脚本 |

## 数据与状态设计

项目采用本地优先的数据组织方式：

- `data/formulas.json`：公式数据源，承载名称、分类、关键词、解释和适用场景。
- `data/strategy-cases.json`：闯关题库，承载解题策略训练材料。
- 本地缓存：保存收藏、重点、复习记录、最近练习、闯关进度和设置。

这种方案减少了后端依赖，适合低成本学习工具的早期验证，也方便通过静态数据持续扩充内容。

## 主要页面链路

- 首页：承载学习入口和核心路径分发。
- 公式列表 / 公式详情：用于浏览、筛选、阅读和收藏公式。
- 搜索页：支持公式名、关键词、别名和拼音首字母检索。
- 闪卡页：用于复习和自评熟练度。
- 闯关页：以策略解题方式连接公式与题目。
- 我的页：汇总收藏、重点、最近练习和薄弱项。

## 构建与验证

当前工程保留以下脚本：

```bash
npm run type-check
npm run build:h5
npm run build:app
npm run build:apk:local
```

已沉淀的验证材料包括：

- `docs/page-entry-convention.md`：页面入口和包装约定。
- `docs/manual-smoke-checklist.md`：手工冒烟检查清单。
- `docs/android-build.md`：Android 本地构建说明。
